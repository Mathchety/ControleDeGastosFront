# Assets do Finansync

## Arquivos Necessários

Este projeto precisa dos seguintes arquivos de imagem:

### Obrigatórios:
- `icon.png` - Ícone do app (1024x1024px)
- `splash.png` - Tela de splash (1242x2436px)
- `adaptive-icon.png` - Ícone adaptativo Android (1024x1024px)
- `favicon.png` - Favicon para web (48x48px)

## Como Criar os Assets

### Opção 1: Usar o Expo Asset Generator
```bash
npx expo-asset-generator
```

### Opção 2: Criar Manualmente
Use qualquer editor de imagens para criar:
- **icon.png**: 1024x1024px, PNG com transparência
- **splash.png**: 1242x2436px, PNG com fundo #667eea
- **adaptive-icon.png**: 1024x1024px, PNG com transparência
- **favicon.png**: 48x48px, PNG

### Opção 3: Download de Templates
Baixe templates gratuitos de:
- https://www.figma.com/community
- https://www.canva.com
- https://icons8.com
